# Sample App Skeleton / 示例应用骨架

This folder provides a non-production sample architecture showing how a new app can adopt `DesignSystemForIOS` from day one.

该目录提供一个非生产示例架构，用于演示新项目如何从 Day 1 接入 `DesignSystemForIOS`。

## Scope / 范围

1. App root policy wiring (theme/style/accessibility/runtime capability)
2. Basic screens (list, form, feedback)
3. No business domain, networking, storage, or analytics coupling

1. 应用根策略接入（theme/style/accessibility/runtime capability）
2. 基础页面骨架（list、form、feedback）
3. 不包含业务域、网络、存储、埋点耦合
